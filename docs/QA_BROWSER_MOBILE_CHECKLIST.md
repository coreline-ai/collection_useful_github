# Browser / Mobile QA Checklist

업데이트 기준: 2026-02-17

## 1) 브라우저 호환성

- [ ] Chrome 최신 버전
- [ ] Edge 최신 버전
- [ ] Firefox 최신 버전
- [ ] Safari 최신 버전(macOS/iOS)

## 2) 공통 핵심 플로우

- [ ] 탭 전환: 통합검색/깃허브/유튜브/북마크
- [ ] 카드 추가/삭제/이동
- [ ] 카테고리 생성/이름변경/삭제
- [ ] 동기화 상태 배지 갱신
- [ ] 검색 및 페이지네이션
- [ ] 다크/라이트 테마 전환

## 3) 모바일 점검

- [ ] 360x800 해상도에서 레이아웃 깨짐 없음
- [ ] 툴팁 탭 동작(hover 없는 환경)
- [ ] 입력창 포커스 시 키보드 오버랩 없음
- [ ] 카드 액션 버튼 터치 영역 충분
- [ ] 모달 포커스/닫기 동작 정상

## 4) 접근성/성능

- [ ] 키보드-only 탭 이동 가능
- [ ] 모달 포커스 트랩/ESC 닫기
- [ ] Lighthouse Accessibility 경고 치명도 0
- [ ] CLS/INP/LCP 급격한 회귀 없음
