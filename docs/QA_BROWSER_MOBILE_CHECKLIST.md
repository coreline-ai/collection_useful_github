# Browser / Mobile QA Checklist

업데이트 기준: 2026-02-17

## 1) 브라우저 호환성

- [ ] Chrome 최신
- [ ] Edge 최신
- [ ] Firefox 최신
- [ ] Safari 최신(macOS/iOS)

## 2) 공통 핵심 플로우

- [ ] 탭 전환: 통합검색/깃허브/유튜브/북마크
- [ ] 카드 추가/삭제/이동
- [ ] 카테고리 생성/이름변경/삭제
- [ ] 동기화 상태 배지 + 마지막 성공 시각 표시
- [ ] 검색/페이지네이션
- [ ] 다크/라이트 테마 전환

## 3) GitHub 전용

- [ ] 상세 모달(Overview/README/Activity) 렌더
- [ ] 메모 입력/저장/복원
- [ ] 요약 재생성 상태(대기/생성중/완료/실패)
- [ ] 요약 툴팁 노출(hover/focus)

## 4) YouTube/Bookmark 전용

- [ ] 요약 재생성 동작
- [ ] 요약 툴팁 노출(hover + 모바일 tap)
- [ ] 링크 열기/추가일 레이아웃 확인
- [ ] Bookmark 중복 정리 도우미 동작

## 5) 모바일 점검

- [ ] 360x800 레이아웃 깨짐 없음
- [ ] 툴팁 tap 동작 정상
- [ ] 입력창 포커스 시 키보드 오버랩 없음
- [ ] 액션 버튼 터치 영역 충분

## 6) 접근성/성능

- [ ] 키보드-only 탐색 가능
- [ ] 모달 포커스 트랩/ESC 동작
- [ ] Lighthouse Accessibility 치명 이슈 0
- [ ] CLS/INP/LCP 회귀 없음
